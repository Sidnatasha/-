# Увеличение размера изображения

**Описание:**
Данный проект представляет собой реализацию алгоритма для увеличения размера цифровых изображений. Используя библиотеки OpenCV и NumPy, мы можем повысить разрешение изображений, сохраняя при этом приемлемое качество.

**Установка:**
```bash
git clone [https://github.com/Sidnatasha/-.git]
cd image-upscaler
pip install -r requirements.txt
